# Promptale

> 당신의 AI 대화가 들려주는 이야기

Claude Code, Gemini CLI 등 AI 코딩 어시스턴트와 나눈 대화를 재미있게 회고하는 웹 앱.

## Features

- **Drag & Drop** — `.jsonl` 세션 파일을 드래그해서 바로 분석
- **활동 히트맵** — GitHub 스타일 일별 활동 시각화
- **시간대별 활동** — 언제 가장 활발히 코딩했는지
- **워드 클라우드** — 자주 사용한 단어/키워드
- **세션 브라우저** — 대화 내역을 쉽게 탐색
- **토큰 & 모델 통계** — 사용량 한눈에 파악

## Quick Start

```bash
npm install
npm run dev
```

브라우저에서 `~/.claude/projects/` 폴더의 `.jsonl` 파일을 드래그하세요.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS v4
- Lucide Icons
