# Competitive Analysis

## 시장 현황

AI 코딩 어시스턴트 대화 히스토리 뷰어는 이미 활발한 생태계.
대부분 **실용적/기능적** — "재미있는 회고" 영역은 비어있음.

---

## 주요 경쟁자

### 1. CCHV (claude-code-history-viewer) — 시장 리더
- **Stars**: 890+
- **기술**: Rust + Tauri v2 (데스크톱 앱)
- **강점**: 7개 프로바이더 지원, 헤드리스 서버 모드, 접근성 최강, 토큰 비용 분석
- **약점**: 데스크톱 설치 필요, "재미" 요소 없음

### 2. Claude Inspector
- **강점**: 브라우저에서 세션 실행/재개, SQLite 인덱싱, 실시간 스트리밍
- **약점**: Claude 전용, 분석/통계 약함

### 3. claude-history (raine)
- **강점**: 터미널 TUI, 퍼지 검색 최강, 세션 Fork 기능
- **약점**: GUI 없음, 터미널 사용자 한정

### 4. SpecStory
- **강점**: 크로스 플랫폼 (Cursor, Copilot, Claude), 자동 마크다운 저장, 공유 URL
- **약점**: 분석 기능 약함, VS Code 확장 한정

### 5. 기타
- **claude-devtools**: DevTools 스타일 세션 분석
- **claude-JSONL-browser**: 웹 기반 JSONL→마크다운
- **Simon Willison의 Timeline**: 단일 페이지 타임라인 도구

## Promptale의 포지셔닝

```
  실용적 ◄─────────────────────────────────► 재미있는
           CCHV   Inspector   claude-history        Promptale
           SpecStory                                   ↑
                                                  여기가 비어있다!
```

### 핵심 차별화 포인트

1. **Promptale Wrapped** — Spotify Wrapped를 AI 코딩에 적용. SNS 바이럴 가능.
2. **세션 리플레이** — 대화를 영상처럼 재생. 아무도 안 하고 있음.
3. **업적/뱃지** — 게이미피케이션. 코딩을 게임처럼.
4. **코드 진화 트래커** — 파일이 어떻게 변했는지 필름 스트립.
5. **웹 전용** — 설치 없이 바로 사용. 공유 링크로 즉시 접근.
